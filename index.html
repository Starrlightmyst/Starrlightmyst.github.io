Web & Print Designer Lavita Starr

アメリカ出身、現在千葉県在住のデザイナー、ラビタ　スターです。日本は5年目ですが日本語を流暢に話せます。オンラインスクールでWeb制作について学んでいます。Webデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

My Skills

Web

HTML/CSSでのコーディングができます。JavaScriptも現在勉強中です。

Illust

Print
